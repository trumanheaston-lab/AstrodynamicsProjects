## Projects

### 🛰️ SSO Mirror Tracker
**Computes the exact inclination required for a sun-synchronous orbit at any altitude and generates a 3D ECI visualization.**

#### What it does
Earth observation satellites need consistent lighting conditions to capture comparable imagery over time. Sun-synchronous orbit (SSO) achieves this by exploiting Earth's equatorial bulge — the J2 perturbation — to precess the orbital plane at exactly the rate Earth orbits the Sun (~1°/day).

This tool solves for the inclination that produces that precession rate at a given altitude, then visualizes the full orbit in the Earth-Centered Inertial (ECI) frame.

#### The physics
The J2 nodal regression equation governs how Earth's oblateness causes an orbit's right ascension of ascending node (RAAN) to drift over time:

```
dΩ/dt = -1.5 * n * J2 * (R_E/a)^2 * cos(i) / (1 - e^2)^2
```

Setting this equal to Earth's mean motion around the Sun (OMEGA_DOT_TARGET ≈ 1.991 × 10⁻⁷ rad/s) and solving for inclination gives the SSO condition. For a circular orbit at 500 km altitude, the required inclination is **97.40°** — slightly retrograde, matching published values.

The 3D trajectory is generated by rotating through the perifocal frame using the full Direction Cosine Matrix (DCM) transformation.

#### Sample output
```
============================================================
============ REFLECT ORBITAL: SSO MIRROR TRACKER ===========
============================================================
Calculated inclination for SSO at 500.0 km altitude: 97.40 degrees
Orbital Period: 94.56 minutes
Passes around Earth per day: 15.2
```

#### How to run
```bash
pip install numpy matplotlib
python SSO_Mirror_Tracker.py
```

### 🚀 LEO to GEO Transfer Calculator
**Computes the delta-v budget for a Hohmann transfer from Low Earth Orbit to Geostationary Orbit.**

#### What it does
Every commercial GEO communications satellite executes this maneuver. The tool calculates the two impulsive burns required — raising apogee to GEO altitude, then circularizing — and outputs the total delta-v budget.

#### The physics
Uses the vis-viva equation at each orbital altitude:

```
v = sqrt(MU * (2/r - 1/a))
```

Two burns computed: Δv₁ at LEO perigee to raise apogee to 35,786 km, Δv₂ at GEO apogee to circularize. Assumes two-body dynamics and instantaneous (impulsive) burns.

#### How to run
```bash
python LEO_GEO_Transfer.py
```

- Add finite burn losses for more realistic delta-v estimates
